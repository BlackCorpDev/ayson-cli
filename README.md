# <p align="center">⚡ AYSon CLI</p>

<p align="center">

</p>

🚀 Features

- ⚡ Fast processing
- 🖥️ Command-line interface
- 🪶 Lightweight
- 🔒 Local processing
- 🚫 No dedicated backend
- ✅ AY.Live support
- ✅ OUO.io support

---

📦 Installation

git clone https://github.com/BlackCorpDev/ayson-cli.git
cd ayson-cli
pip install -r requirements.txt

---

▶ Usage

python main.py "https://ay.live/xxxxxxxx"

Example:

python main.py "https://ouo.io/xxxxxxxx"
